# Unreasonable fiction

Этот репозиторий служит убежищем для моих заметок / рассказов / стихов и т.д.

Почему гит? Да просто, чтобы не потерялось. А если я когда-нибудь не смогу найти доступ в свой гит, то дальше писать мне уже точно не стоит

## Структура проекта

### ├ [`{введите_ваше_название}`](title_goes_here/README.md)

Откровенно бессмысленная повесть. Весь смысл которой в её бессмысленности. Все фрагменты осмысленности случайны

### ├ [`{ВведитеВашеНазвание}`](TitleGoesHere/README.md)

Всё то же самое, что и в [`{введите_ваше_название}`](title_goes_here/README.md). Но начиная с 3-ей главы пишется ИИ. Что по задумке должно поднять градус бессмысленности до запредельных величин

### └ [`Через конечность: Эпоха приверженных`](through_the_finite/README.md)

Просто безобидный стеб над пафосными и тупыми боевиками. Задача книги — держать достойный уровень безмуной крутости на пиксель на протяжении всей себя
